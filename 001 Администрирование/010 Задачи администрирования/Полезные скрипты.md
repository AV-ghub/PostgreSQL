## Настройки
#### Вывод текста системной функции
```
-- psql
\set ECHO_HIDDEN on
\dt employees
\unset ECHO_HIDDEN
```
#### Приведение к типу regclass
```
select * from pg_attribute where attrelid = 'employees'::regclass
где
'employees'::regclass = (select oid from pg_regclass where relname = 'employees')
```
